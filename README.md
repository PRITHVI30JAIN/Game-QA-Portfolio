# 🎮 Game QA Portfolio — Prithvi Jain

## 📌 Overview

This repository showcases my hands-on Game Quality Assurance work focused on Rockstar titles including GTA V and Red Dead Redemption 2.

As a BCA final-year student and active player of these titles, I approached gameplay with a QA mindset — systematically identifying, reproducing, and documenting defects using structured testing practices similar to industry workflows.

---

## 🧪 Projects Included

### 🔹 Bug Reports

* Logged 30+ reproducible defects across GTA V and RDR2
* Each report includes:

  * Steps to Reproduce (STR)
  * Expected vs Actual Results
  * Severity classification (Critical / Major / Minor / Cosmetic)
  * Patch tracking status (Open / Fixed / Regressed)

---

### 🔹 Test Case Design

* Designed 40+ structured test cases for GTA V Heist missions
* Coverage includes:

  * Mission trigger logic
  * Save/load integrity
  * Multiplayer synchronization
  * UI/HUD validation
  * Failure-state handling

---

### 🔹 Cross-Platform Analysis

* Conducted structured testing of GTA V across PC and PS4
* Compared:

  * Performance (load times, frame rate behavior)
  * Input latency
  * Rendering differences (textures, pop-in)
  * UI consistency across platforms
